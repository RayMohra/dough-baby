## A really easy setup

You'll need [git](https://git-scm.com), [nodejs](https://nodejs.org/en/) and [npm](https://docs.npmjs.com) installed. Then you'll need to perform these steps.

```sh
git clone git@github.com:meanjim/dough.git
cd dough
npm install -g gulp
npm install
npm start
```
